Introduction

Bienvenue dans la documentation de pcUtils. Cet outil est conçu pour simplifier et automatiser diverses tâches d'administration système grâce à un système de plugins modulaires et une interface utilisateur Textual.

Cette documentation vous guidera à travers les différents aspects de l'application :

  • Comprendre l'architecture globale.
  • Apprendre à créer vos propres plugins pour étendre les fonctionnalités.
  • Maîtriser l'exécution à distance de plugins via SSH.
  • Utiliser les séquences pour regrouper des tâches complexes.
  • Simplifier la configuration avec les templates.
  • Naviguer et comprendre les différents écrans et composants de l'interface.

Utilisez la navigation rapide ou parcourez les sections ci-dessous pour accéder aux informations détaillées.
